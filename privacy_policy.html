Privacy Policy

Last updated: [当前日期]

1. Information Collection

The Web QR Code Generator Chrome extension does not collect, store, or transmit any personal information or browsing data. The extension only processes the current webpage URL locally to generate QR codes.

2. Data Usage

The only data the extension uses is:

  • Current webpage URL - to generate the QR code
  • Website favicon - to display in the QR code
  • Page title - to display below the QR code

All processing is done locally in your browser.

3. Data Storage

The extension does not store any data permanently. All information is temporary and is cleared when you close the QR code display.

4. Third-Party Services

The extension uses the following third-party services:

  • QR Server API - only for generating QR codes
